# Smart Parking Sensor System

## Description
This project uses an ultrasonic sensor to measure distance and help drivers park safely using LEDs and a buzzer.

## Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- 3 LEDs (Red, Yellow, Green)
- Resistors (470Ω)
- Buzzer
- Breadboard and wires

## How it Works
The ultrasonic sensor measures distance. Depending on the distance:
- Safe (>50cm): Green LED
- Caution (20–50cm): Yellow LED + slow beep
- Danger (<20cm): Red LED + fast beep
- Critical (<5cm): All LEDs + continuous sound
